This is the first public version of my Sins of a Solar Empire 2 Compendium.
It consists in an HTML file based on the game data (entities) and uses in game assets for illustrations.

The file is large as it embeds low quality images. As a result, it may take a minute or two to load the first time (then no more loading times are needed).
You may want to enhance these images by pointing to your local game texture folder on your hadr drive so these are replaced for a nicer experience.
